# _acc_rate

Get or set the rate of acceleration in the speed-profile.

### Description

- unit : %
- range : 1 to 100
- default value : 100

### Syntax

```python
var res
res = _acc_rate
```

### Sample

```python
   ...
   # Print current accel-rate, and set to 70%.
   print _acc_rate
   _acc_rate=70
   ...
   end
```
