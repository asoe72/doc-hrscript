# _dec_rate

Get or set the rate of deceleration in the speed-profile.

### Description

- unit : %
- range : 1 to 100
- default value : 100

### Syntax

```python
var res
res = _dec_rate
```

### Sample

```python
   ...
   # Print current decel-rate, and set to 70%.
   print _dec_rate
   _dec_rate=70
   ...
   end
```
